# Use Cases

This section contains AI use cases written to be practical, cross-functional, and easy to evaluate.

Each use case includes:
- Problem framing and workflow fit
- Data needs and quality considerations
- Approach options (rules, retrieval, ML, agents, or hybrids)
- Trust and guardrails
- Adoption considerations
- Success metrics
